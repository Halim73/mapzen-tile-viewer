The config.txt file should have a comma separated list for the zoom levels ip address
followed by a newline and a colon separated list of zoom levels in the same order as the given ip addresses
followed by a newline and a comma separated list for the number of clients and the view radius and wait seconds and loop count
eg
    54.244.181.20
    10
    1,3,5,5
to run
    java -jar Tester.jar
